# Sources

Each skill in this library is built from published sources—books, courses, guides—extracted into execution-level principles.

This is the **source-first** philosophy: every principle traces to a published expert. Nothing invented.

## How Sources Work

A source file isn't a summary. It's a translation of theory into execution:

- **What it is** — the mechanism
- **How it applies** — specific contexts (landing pages, emails, ads, etc.)
- **What Claude gets wrong by default** — the failure mode to override

The skills embed these principles inline. The source files provide deeper reference when you need it.

## What's Included Here

This public repo includes one example source:

```
sources/books/schwartz/breakthrough-advertising/
```

Eugene Schwartz's *Breakthrough Advertising* — awareness levels, market sophistication, mass desire channeling. One of the foundational texts for direct-response copy.
